# RobertBot V2 — Design Lab 🎨 (SAFE MODE)

> מעבדת עיצוב **מבודדת לחלוטין**. אפס תלות במערכת הקיימת, אפס נגיעה בפרודקשן.
> כל הקבצים כאן הם HTML/CSS סטטי — **פותחים בלחיצה כפולה בדפדפן**, בלי build ובלי שרת.

## איך פותחים
לחיצה כפולה על כל קובץ `.html`, או גרירה לדפדפן. הכי טוב להתחיל מ-`design-system.html`.

## מה יש כאן
| קובץ | מה זה |
|---|---|
| **design-system.html** | ספר המותג החי — כל הצבעים, הטיפוגרפיה והקומפוננטות |
| **index.html** | דף הבית V2 |
| **onboarding.html** | אשף יצירת בוט V2 (עם מילוי חכם ב-AI) |
| **dashboard.html** | אזור אישי V2 (app shell מלא) |
| `assets/tokens.css` | טוקני העיצוב — מקור האמת היחיד לצבעים/טיפו/spacing/radius/shadow |
| `assets/lab.css` | ערכת הקומפוננטות המשותפת לכל המסכים |
| `audit/01-product-audit.md` | אודיט המוצר הקיים (Phase 1) |
| `audit/02-ux-rebuild.md` | מסע המשתמש החדש + הפחתת חיכוך (Phase 5) |
| `audit/research/greeninvoice.md` | מחקר Green Invoice (Phase 2) |
| `audit/before/` · `audit/after/` | צילומי מסך להשוואה |
| `../PRESENTATION.md` (בתוך design-lab) | המצגת המלאה |
