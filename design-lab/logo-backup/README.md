# גיבוי לוגו ישן — Robert

תיקייה זו שומרת את הלוגו/אייקונים שהיו בשימוש לפני המעבר ללוגו ה-Waveform (גלי הקול).
אם תתחרט — אפשר לשחזר מכאן.

## מה שמור כאן
| קובץ | מה זה | היכן היה בשימוש |
|---|---|---|
| `favicon-old.svg` | האייקון הישן (ריבוע כחול + "R" סריף + נקודה ירוקה) | `app/public/favicon.svg` (לשונית הדפדפן) |
| `logomark-old.svg` | הסמל הישן (עיגול + חץ) | login / dashboard / onboarding / reset-password |

## איך לשחזר (אם מתחרטים)
1. להעתיק את `favicon-old.svg` בחזרה אל `app/public/favicon.svg`.
2. להחזיר את ה-`LogoMark` (עיגול+חץ) בארבעת קבצי העמודים, או פשוט לבקש: "תחזיר את הלוגו הישן".
3. commit + push (deploy) באישור הבעלים.

> נוצר אוטומטית במעבר ללוגו החדש. אל תמחק אלא אם ההחלטה על הלוגו החדש סופית.
